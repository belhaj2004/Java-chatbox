# Java Chatbox

A desktop chat application built with Java Swing featuring a custom UI with animations and sound effects.

## Features
- Real-time message board
- Custom color-coded cells
- Sound effects on interactions
- Animated GIF responses
- Custom logo and branding

## Tech Stack
![Java](https://img.shields.io/badge/Java-ED8B00?style=flat&logo=openjdk&logoColor=white)

## 🚀 How to Run
```bash
javac MessageBoard.java
java MessageBoard
```
