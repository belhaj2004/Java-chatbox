# JavaFX Message Board

A desktop-based JavaFX message board application featuring an interactive graphical interface, multimedia integration, animated responses, and user-driven interaction.

## Demo Video

🎥 [Watch the Demo on YouTube](https://www.youtube.com/shorts/cuESthr3tY0)

## About

This project is a JavaFX desktop application developed to simulate an interactive message board experience. The application combines graphical user interface components, animated GIF responses, image assets, and sound effects to create a more engaging user interaction flow.

Although the repository name includes “java-swing-chatbot,” this project is built using **JavaFX**.

---

## Features

- JavaFX-based desktop GUI
- Interactive message board system
- Animated GIF responses
- Image integration
- Sound effect support
- Event-driven user interaction
- Custom visual design
- Beginner-friendly Java GUI project

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/belhaj2004/java-swing-chatbot.git
```

### 2. Open the project folder

```bash
cd java-swing-chatbot
```

### 3. Install JavaFX

Download JavaFX SDK from:

```text
https://gluonhq.com/products/javafx/
```

Recommended version:

```text
JavaFX 17 LTS
```

---

### 4. Compile the project

Replace `PATH_TO_JAVAFX_LIB` with your own JavaFX `lib` folder path.

```bash
javac --module-path "PATH_TO_JAVAFX_LIB" --add-modules javafx.controls MessageBoard.java
```

---

### 5. Run the application

```bash
java --module-path "PATH_TO_JAVAFX_LIB" --add-modules javafx.controls MessageBoard
```
